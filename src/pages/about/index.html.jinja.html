MOOOOO.ooo is the home of most of my side projects that I find suitable for sharing. It also contains solutions to problems that I was unable to previously find information for online. The focus of these topics is primarily computing and engineering.

The website is hosted with the aid of the InterPlanetary File System (IPFS) - a peer-to-peer technology that aims to decentralize the web while simultaneously making it faster, safer and more permanent. In the event that the domain name expires or becomes inaccessible, the website may be loaded through the IPFS portal via this link. The source code for the website can be found here.

{{ heading("About the Owner") }}

My name is Colin Wallace. I write music and contribute to open source coding projects in my free time. Some might describe me as a Linux enthusiast (I even belong to that rare breed of Linux gamers). I'm a privacy/free speech proponent and supporter of the EFF and other organizations that work to protect our civil liberties. I hold a BS in Electrical Engineering from the University of Washington and am currently living in Seattle.

Some notable projects to which I have contributed include:

  • Rust - a systems programming language
  • LMMS - an open source Digital Audio Workstation
  • Stepmania - a music/rhythm game (Dance Dance Revolution clone)
  • Printipi - a complete firmware package for 3d printers

More details can be found on my Github page or through my {{ onsite_link("resume.pdf"|path_from_here, "resume") }}. Feel free to reach out to me for any reason (except to advertise your product) - my email can be found at the bottom of this page.
